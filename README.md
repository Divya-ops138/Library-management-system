# Library Management System (LMS)

A simple Library Management System built using C++ with a web-based interface, MySQL database, and file storage.  
The system supports book management, member management, and issuing/returning books.

## Features
- Add, update, delete books
- Search books
- Add & manage members
- Issue and return books
- Track due dates
- MySQL + file storage support
- Web-based UI (HTML/CSS/JS)

## Technologies Used
- C++
- HTML, CSS, JavaScript
- MySQL Database
- File handling

## How to Run
1. Import `schema.sql` into MySQL.
2. Compile C++ code using g++.
3. Place the compiled file inside your server's `cgi-bin` folder.
4. Open `index.html` from the `web` folder in a browser.
